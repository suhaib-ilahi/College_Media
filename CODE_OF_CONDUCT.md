# Code of Conduct

## Our Pledge

College Media is a community-driven project built for college students and developers.  
We are committed to fostering a welcoming, inclusive, and respectful environment for everyone who contributes to or interacts with this project.

Participation in this project means agreeing to uphold this Code of Conduct in all project spaces.

---

## Our Values

We value:
- Respectful and constructive collaboration
- Inclusivity across gender, race, caste, religion, nationality, sexual orientation, disability, and background
- Open discussion and healthy disagreement
- Accessibility-first and user-centric development
- Learning, mentorship, and growth

This project is built by students, for students, and everyone deserves to feel safe and heard.

---

## Expected Behavior

All contributors and participants are expected to:

- Be respectful and considerate in language and actions
- Provide constructive feedback and accept criticism gracefully
- Use inclusive and welcoming language
- Focus discussions on improving the project
- Be mindful of accessibility and usability
- Respect differing viewpoints and experiences
- Follow project contribution guidelines and maintain code quality

---

## Unacceptable Behavior

The following behaviors will not be tolerated:

- Harassment, discrimination, or hate speech of any kind
- Personal attacks, insults, or derogatory comments
- Sexist, racist, casteist, homophobic, or transphobic language
- Trolling, spamming, or deliberately disruptive behavior
- Doxxing or sharing private information without consent
- Threats, intimidation, or encouragement of violence
- Sexualized language or imagery
- Misuse of the platform to spread misinformation or harmful content

Intent does not outweigh impact.

---

## Scope

This Code of Conduct applies to all project spaces, including but not limited to:

- GitHub issues, pull requests, and discussions
- Code reviews and comments
- Documentation and community discussions
- Any public or private communication related to the project

It also applies when individuals are representing the project in public spaces.

---

## Reporting Issues

If you experience or witness behavior that violates this Code of Conduct, please report it.

You can report issues by contacting the project maintainers:

- Maintainer: @janavipandole  
- Original Creator: @abhishekkumar177  

All reports will be handled confidentially and without retaliation.

---

## Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Depending on the severity, actions may include:
- A private warning
- Temporary restriction from participation
- Removal of comments or contributions
- Permanent ban from the project
- Escalation to GitHub moderation if necessary

All enforcement decisions will be made in good faith to protect the community.

---

## Accessibility & Inclusion Commitment

College Media prioritizes:
- WCAG-compliant UI and accessible components
- Inclusive language in code, UI, and documentation
- Respect for users from diverse academic, cultural, and socioeconomic backgrounds

Contributors are encouraged to raise accessibility concerns proactively.
